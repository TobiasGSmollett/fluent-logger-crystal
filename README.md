# fluent-logger-crystal

A Fluentd client for Crystal.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  fluent-logger-crystal:
    github: TobiasGSmollett/fluent-logger-crystal
```

## Usage

```crystal
require "fluent-logger"
```
